**Story**: 3.5 - Plain Text Output Format for LLM Upload

**Status**: Ready for review

## Test Objective

Validate that TXT output can be copied and pasted directly into ChatGPT and Claude chat interfaces without any manual cleanup.

## Prerequisites

- [x] All 41 automated tests passing
- [x] Sample TXT outputs generated in `docs/examples/txt-output-samples/`
- [ ] Access to ChatGPT (GPT-4 or GPT-3.5)
- [ ] Access to Claude (Claude 3.5 Sonnet or Claude 3 Opus)

## Test Procedure

### Step 1: Generate Test Output
```bash
# Already complete - use sample files in docs/examples/txt-output-samples/
# sample-basic.txt
# sample-with-metadata.txt
```

### Step 2: ChatGPT Validation

1. Open ChatGPT (https://chat.openai.com)
2. Open `docs/examples/txt-output-samples/sample-with-metadata.txt`
3. Select all (Ctrl+A / Cmd+A)
4. Copy (Ctrl+C / Cmd+C)
5. Paste into ChatGPT message box
6. Send message: "Summarize the content of these chunks"
7. **Validate**:
   - [ ] No error messages about formatting
   - [ ] ChatGPT recognizes chunk delimiters
   - [ ] ChatGPT understands metadata headers
   - [ ] Summary is coherent and accurate
   - [ ] No mention of markdown/HTML artifacts

### Step 3: Claude Validation

1. Open Claude (https://claude.ai)
2. Open `docs/examples/txt-output-samples/sample-with-metadata.txt`
3. Select all (Ctrl+A / Cmd+A)
4. Copy (Ctrl+C / Cmd+C)
5. Paste into Claude message box
6. Send message: "Analyze the structure and content of this document"
7. **Validate**:
   - [ ] No error messages about formatting
   - [ ] Claude recognizes chunk delimiters
   - [ ] Claude understands metadata headers
   - [ ] Analysis references entity IDs correctly
   - [ ] No mention of formatting artifacts

### Step 4: Edge Case Validation

Test with `sample-basic.txt` (no metadata):
- [ ] ChatGPT handles clean text without metadata
- [ ] Claude handles clean text without metadata

Test with `sample-custom-delimiter.txt`:
- [ ] Both LLMs recognize custom delimiter pattern
- [ ] Chunk boundaries correctly identified

## Success Criteria

✅ **PASS** if:
- ChatGPT and Claude accept all 3 sample files without errors
- No manual cleanup required (no removing markdown, HTML, etc.)
- LLMs correctly parse chunk structure
- LLMs understand and use metadata when present
- Summaries/analyses are coherent and accurate

❌ **FAIL** if:
- Error messages about formatting
- LLMs confused by artifacts
- Manual cleanup required before upload
- Chunk delimiters not recognized
- Metadata headers cause parsing issues
